# Ringtone file
Place your ringtone.mp3 file here for incoming doctor calls.

Recommendations:
- File name: ringtone.mp3
- Duration: 2-5 seconds (will loop)
- Format: MP3 or OGG
- Volume: Medium (the code applies 0.7 volume)

Free ringtone sources:
- Zapsplat.com
- Freesound.org
- Mixkit.co

If no file is provided, a simple beep tone will be generated automatically.
